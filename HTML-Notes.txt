HTML - hyper text markup language
->html is not case sensitive.

HTML is the code that is used to structure a web page and its content.moreover its like a skeleton of a body.

Basic HTML Page
<!DOCTYPE html>
<html> ///root of an html document
 <head>  /// container for metadata
 <title>My First Page</title> ///page title will be displayed
 </head>
 <body> ///contains all data rendered by the browser
 <p>hello world</p>  ///paragraph tag
 </body>  
</html>


->Most of html elements have opening & closing tags with content in between
->Some tags have no content in between, eg - <br>, <hr>

->HTML is NOT case sensitive
